Este proyecto consiste en la reutilización de hardware de legado para crear un sistema de almacenamiento conectado a red (NAS) robusto.

### Tecnologías utilizadas:
- **SO:** Ubuntu Server
- **Software:** Nextcloud
- **Hardware:** Computadora de escritorio optimizada

### Logros:
- Centralización de archivos personales.
- Acceso remoto seguro.
- Configuración de backups automatizados.
